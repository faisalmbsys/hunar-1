Hunar is a luxury architectural design studio platform that bridges the gap between digital precision and physical spatial aesthetics. A sister brand to MBSYS, Hunar focuses on minimalist elegance and architectural storytelling.

---

## 🎨 Design Philosophy

- **Architectural Minimalism**: A clean, spacious layout using high-contrast typography (Inter/Outfit) and curated negative space.
- **Cross-Brand Synergy**: Shares a unified transition engine with MBSYS, allowing users to jump between Infrastructure and Design without breaking immersion.
- **Bespoke Portfolio Presentation**: Custom-built gallery components optimized for high-resolution spatial photography.
- **Glassmorphic Navigation**: Subtle blur effects and backdrop filters for a modern, airy interface.

## ⚡ Core Technologies

- **Vite & React**: Lightning-fast Hot Module Replacement (HMR) and optimized build outputs.
- **Tailwind CSS v4**: Leveraging the latest CSS-in-JS performance and utility first workflow.
- **Framer Motion**: Orchestrating complex entry animations and scroll-triggered reveals.
- **Lucide Icons**: Consistent, scalable vector iconography for a sharp UI.

## 🛠️ Getting Started

### Installation
1. Clone the repository
2. Navigate to the project directory:
   ```bash
   cd Hunar
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```

## 🌐 Infrastructure

Deployed via **Netlify** on a dedicated subdomain (`hunar.mbsys.co.in`), fully integrated with the main MBSYS DNS zone for unified SSL management.
